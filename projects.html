  • CANVAS Small Satellite

    Instrumentation & Data Validation

  • PI: Dr. Robert Marshall (University of Colorado Boulder)

    PROJECT ROLE: Instrumentation Lead | FPGA Testing & Validation

    PROJECT DESCRIPTION: First involved in writing Test scripts to model the FPGA, using fast Fourier transforms (FFTs) to reduce the data collected from the satellite. Parse packets of data into science data for data analysis and validate with instrument results. Testing the B-Field and E-Field instrument with Frequency Response Analysis, Sensitivity and Cross Talk. Proceeding with Day-in-the-Life test to test instrument and algorithm using signal generators to simulate upper atmosphere signals.

    PROJECT SKILLS: Electromagnetic Waves, FPGA Algorithm, Embedded Systems, Python, Signal Processing, Data Analysis, Instrument Testing

  • ELYSIUM Mission Design

    Command & Data Handling/ Flight Software

  • Course: ASEN 5148: Spacecraft Design (University of Colorado Boulder)

    PROJECT ROLE: Command & Data Handling Subsystem Lead

    PROJECT DESCRIPTION: Our team led the design and development of an autonomous satellite mission with a dual purpose: assist the Spitzer Space Telescope and search for Near Earth Objects (NEOs). Through rigorous reviews (CoDR and PDR), we refined our mission concept. Our satellite autonomously locates and assists Spitzer, acting as a data relay conduit. My role focused on the Command & Data Handling Subsystem, where I curated components, designed wiring interfaces, prepared a data budget and co-led the Flight Software System using NASA CFS. This project exemplifies our commitment to space exploration and cutting-edge technology.

    PROJECT SKILLS: Autonomous in-Space servicing, NASA CFS, Embedded Systems, Space Mission Design

  • MARS EXPRESS

    Analysis of Ionospheric data from MARSIS

  • Course: ASEN 5245: Radar and Remote Sensing (University of Colorado Boulder)

    PROJECT TITLE: Analysis of Ionospheric data from MARSIS

    PROJECT DESCRIPTION: Utilized data from MARSIS (Mars Advanced Radar for Subsurface and Ionosphere Sounding) to conduct a comprehensive analysis of the Martian ionosphere. Generated ionograms and applied the Chapman profile on ionospheric soundings to derive essential parameters contributing to a deeper understanding of Mars' ionospheric characteristics.

    PROJECT SKILLS: Ionospheric Sounding, Radar Systems, Signal Processing, Planetary Radio Science
